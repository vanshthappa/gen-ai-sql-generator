# 🤖 Gen AI SQL Generator

An AI-powered SQL assistant that allows users to query databases using natural language.  
Built using Python, Streamlit, SQLite, Ollama, and Large Language Models (LLMs).

---

# 🚀 Features

- Convert natural language questions into SQL queries
- AI-powered database analysis
- Interactive Streamlit frontend
- SQLite database integration
- Local LLM support using Ollama
- Schema-aware query generation
- Beginner-friendly interface

---

# 🛠️ Tech Stack

- Python
- Streamlit
- SQLite
- Ollama
- LangChain
- DeepSeek / Qwen / Llama Models
